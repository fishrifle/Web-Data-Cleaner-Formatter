# Python Data Cleaning Automation Tool

A lightweight yet powerful Python automation script that cleans and formats messy CSV data. 
It removes duplicates, normalizes formatting, validates emails, converts dates, and outputs 
a clean dataset ready for use in applications or analysis.

## Features
- Clean & normalize names
- Validate and standardize emails
- Convert dates to ISO format
- Remove duplicate entries
- Prepare output CSV for production use

## Tech Used
- Python 3
- CSV Processing
- Regular Expressions
- Date Parsing

## How to Run

1. Clone the repository:

git clone <your repo>

2. Add your raw CSV file to `data/raw_data.csv`.

3. Run the script:

python scripts/clean_data.py

4. Your cleaned file will appear here:

data/cleaned_data.csv


## Example Use Cases
- Cleaning customer data
- Preparing CSV imports for CRMs
- Standardizing marketing lists
- Removing duplicate leads
- Backend preprocessing for web apps
